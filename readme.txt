# Group Expense Manager

A simple PHP-based web application to **track shared group expenses**. Ideal for roommates, friends, or travel groups who want to easily add, view, and settle transactions together.

## Features

- Manage multiple groups with different users
- Add expenses with amount, date, comment, and group
- Calculate how much each user owes or is owed
- Automatically split and record debts between users
- Tracks whether transactions are "settled" or still active
- Color-coded transactions per user (customizable via JSON)
- User login session management

## Technologies Used

- PHP
- MySQL / MariaDB
- HTML / CSS
